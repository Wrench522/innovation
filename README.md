# Car Innovation Website

A simple static website showcasing innovative cars and automotive technology.

## Features

- Home page with hero image
- Featured cars section
- About section
- Contact form

## How to Run

1. Open `index.html` in your web browser.

For a local server (optional):
- Install Node.js
- Run `npx http-server` in the project directory
- Open the provided URL in your browser.

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Troubleshooting

- Ensure all files are in the same directory.
- If images don't load, check internet connection (using placeholder images).
